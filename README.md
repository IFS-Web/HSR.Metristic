## Development
Watch files and compile TS to JS on changes:
```shell
gulp watch
```
Serve app (compiles on changes and reloads):
```shell
gulp serve
# or
npm start
```
Deploy app to directory `deploy`:
```shell
gulp deploy
```
Access app:
`localhost:8080`
